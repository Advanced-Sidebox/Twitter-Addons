# Advanced Sidebox Twitter-Addons

Addon modules for Twitter integration

These addon modules utilize both [twitter-api-php
](https://github.com/J7mbo/twitter-api-php) and [widgets.js](https://dev.twitter.com/web/javascript) in order to properly display Twitter components.

Due to standard Twitter element sizes, 180px is the smallest column width allowable for these addons— while ASB's default column width is 160px. Twitter modules **will** stretch the side box column if the width is set below 180px.

## Latest Status

Displays the latest status update from the named Twitter account.

This addon requires OAuth credentials.

## Embedded Tweet

Displays an embedded Tweet.

This addon requires OAuth credentials.

## Follow Button

Displays a Twitter Follow Button for the specified account.

## Timeline

Displays a timeline for the specified Twitter account.
